This weeks episode is about "Arulmigu Idaisuranathar Alayam,ThiruIdaichuram".

Location&Direction:This
sthalam can be reached 4kms from Thiruporur which inturn can be reached
enroute chengalpattu-Mahabalipuram busroute.

Current Name : Thiruvadisoolam (ThiruIdaichuram)
MainDeity : Idaichuranatha,Gnanapureeswarar
Godesses : Goparthanaambigai
HolyTree : Iluppai
Theertham : SanathkumaraTheertham

Pathigam : Thirugnanasambantha swamigal

Darshan Time : Morning 8am- 12pm Evening 5.30pm-8pm

Pooja timings : MorningSandhi 8 am,Evening-6pm

Lord shiva Came as "Idaiyan" in disguise and took thirugnana sambantha swamigal to this temple.
